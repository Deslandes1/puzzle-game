Triangle Memory Puzzle

A brain training game that boosts memory and cognitive skills through drag-and-drop puzzles. Perfect for adults and kids who want to improve their memory and learning abilities.

Live Demo: https://puzzle-game-gdcx5vdkwhbbm824cwxcc9.streamlit.app/

About the Project

Each square hides a random number. Drag the triangle with the matching number to the correct square. Remember where each number is hidden to win. Three difficulty levels: 10, 20, and 100 squares. Audio feedback, celebration animations, AI voice description, multilingual support, and responsive design.

Why Play?

Improves short-term memory, concentration, and focus. Develops better learning habits. Fun and engaging for all ages.

How It Works

A grid of squares hides random numbers. Triangles with matching numbers are below. Drag each triangle to its matching square. Correct matches turn green and stay. Wrong matches trigger an error sound. The challenge is remembering where each number is hidden.

Difficulty Levels

Level 1: 10 squares - Beginners and kids
Level 2: 20 squares - Intermediate players
Level 3: 100 squares - Memory experts

Getting Started

Prerequisites: Python 3.8+ and pip.

Clone the repository:
git clone https://github.com/yourusername/triangle-memory-puzzle.git
cd triangle-memory-puzzle

Install dependencies:
pip install -r requirements.txt

Run the app:
streamlit run app.py

Open http://localhost:8501. Login password: 20082010.

Deployment
